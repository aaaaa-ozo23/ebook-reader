# Stage 13.5 metadata editor fidelity

The editor extends the approved bookshelf language with the same warm paper surface, deep-ink
copy, teal actions, amber focus ring, restrained depth, and 44 px primary targets.

Desktop uses a centered modal with a 2:3 cover workspace and field column. At 600 px and below it
becomes a bottom-aligned sheet. Motion is a 170 ms settle and is removed under reduced motion.

Title, author, and cover expose independent **Restore automatic** actions. Custom cover controls
support PNG/JPEG/WebP up to 10 MiB, positioning and zoom, then output normalized 600×900 WebP.
The backend repeats signature, decoding, pixel-count, and type validation.
