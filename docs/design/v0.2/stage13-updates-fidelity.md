# Stage 13.7 updates fidelity

Updates lives inside the existing settings center instead of opening a separate system window.
Desktop retains the deep-ink navigation rail and warm-paper workspace; 375 px keeps the full-screen
sheet and exposes Data & Backup / Updates as compact 44 px navigation targets.

The page makes control boundaries visible: checks are manual by default, daily checks are opt-in,
downloads always require confirmation, and installation is shown as the irreversible boundary.
Idle, checking, up-to-date, available, downloading, downloaded, installing, canceled, signature
failure, and network failure reuse teal progress, amber focus, and restrained error red.

NSIS and MSI are presented as different tracks. NSIS accepts independently signed update artifacts;
MSI disables in-app actions and explains manual over-installation. Reduced motion removes pulsing and
travel while preserving status and progress text.
